# Donocha.tj
Ai-donocha — это современная AI-платформа с интерфейсом в стиле ChatGPT, позволяющая пользователям общаться с искусственным интеллектом в реальном времени.  Проект сочетает в себе минималистичный дизайн, высокую скорость работы и интеграцию с AI API, что делает его удобным инструментом для обучения, работы и генерации контента
